# Movie Genre Classification

## Overview

This project uses Machine Learning and Natural Language Processing (NLP) techniques to classify movies into their respective genres based on movie plot descriptions.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* NLTK
* TF-IDF Vectorization

## Features

* Text Preprocessing
* Feature Extraction using TF-IDF
* Genre Prediction
* Machine Learning Classification
* Model Evaluation
